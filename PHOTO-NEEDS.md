## Delivery convention

- **Format:** WebP, q=80, sRGB
- **Hero (3:2):** master 1600×1067; I'll generate the size variants
- **Vertical (2:3):** 2000×3000
- **Naming:** `{site}-{slug}-hero.webp` — e.g. `boda-jarabacoa-hero.webp`, `inmo-cap-cana-villa-hero.webp`

Drop the files in the respective repo's `public/images/` folder, then ping me to wire them into the data files:

- boda: `lib/weddingCities.ts` (imageSet), `lib/blogPosts.ts` (needs new `hero` field), `lib/articles.ts`
- inmobiliaria: `lib/realEstateCities.ts`, `RealEstateServicePage.tsx`, `RealEstateIndustryPage.tsx`, `RealEstateHome.tsx`
- drone: `lib/droneData.ts`
- estudio + parent: set featuredmedia in WP admin (next `scrape/fetch-wp.mjs` pulls it in) OR add per-slug overrides
